# GitHub Pages Setup Instructions

To set up GitHub Pages for your repository, follow these steps:

1. Go to your repository on GitHub.
2. Click on the `Settings` tab.
3. Scroll down to the `Pages` section.
4. Under `Source`, choose the branch you want to use (usually `main`) and select the folder (root or `/docs`).
5. Click `Save`.
6. Your site will be published at `https://<username>.github.io/<repository>` where `<username>` is your GitHub username and `<repository>` is your repository name.
